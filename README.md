# Contabilidade

Processos, análises e relatórios contábeis da Dorinhos.

## Estrutura

- `CLAUDE.md` — Configuração de skills e preferências
- `MEMORY.md` — Índice de memória
- `memory/glossary.md` — Termos contábeis, processos e padrões brasileiros

## Como Usar

Abra este projeto no Claude Code. Ele carregará automaticamente `CLAUDE.md` com as skills específicas para contabilidade.

## Skills Disponíveis

- **finance:financial-statements** — Preparar demonstrações financeiras
- **finance:journal-entry** — Registrar lançamentos contábeis
- **finance:reconciliation** — Conciliação de contas
- **data:analyze** — Análise de dados contábeis
- **customer-support:kb-article** — Documentar procedimentos

## Padrões

- Normas contábeis brasileiras (IFRS/CPC)
- Foco em precisão e conformidade
- Análise orientada por dados
